Anaconda 설치

현재 tensorflow는 python 3.6까지 지원하므로 anaconda3 - 5.1 버전과 호환되는 것이 확인된다.
다만 다중 커널을 이용할 수 있기에 가장 최신 anaconda를 설치해도 무방하다.
국내 증권사 API 등 32비트만 지원하는 API가 있으므로 사용 목적에 따라 구분해 설치.
anaconda installer archive


tensorflow 사용하기

anaconda prompt>
python -m pip install --upgrade pip
conda create -n tensorflow python=3.5

activate tensorflow
(tensorflow)> pip install tensorflow

확인차:
python
>>> import tensorflow as tf

(tensorflow)> jupyter notebook
...에서는 tensorflow import하다가 에러가 난다.

(tensorflow)> conda install nb_conda
이렇게 하면 notebook의 default가 root에서 tensorflow_env로 바뀐다
이제 jupyter notebook>New>Python[conda env:tensorflow]에서 코드를 입력할 수 있다.
잘 되었는지 확인도 해보자.
import tensorflow as tf
hello = tf.constant("Hello World!")
sess = tf.Session()
print(type((sess.run(hello)).decode(encoding='utf-8)))
sess.close()

progress bar for loop:
conda install tqdm

from tqdm import tqdm_notebook #사용 시엔 tqdm을 불러내서

다중 커널 사용하기

설치된 커널 확인:
anaconda prompt>
jupyter kernelspec list
혹은 jupyter notebook> new 에서도 확인 가능하다.

설치할 버전이 x.y일 때
conda update conda
conda create -n py27 python=x.y anaconda

activate pyxy

ipython kernelspec install-self
deactivate
